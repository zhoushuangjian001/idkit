## Introduce
This function is mainly to directly convert hexadecimal strings into colors.
## Example Display
```dart
// HEX
Color color = '#3767FE'.hexColor;
color = '#3767FE'.color;
color = '3767FE'.hexColor;
color = '3767FE'.color;
// AHEX
color = '#FF3767FE'.aHexColor;
color = 'FF3767FE'.aHexColor;
// HEXA
color = '#3767FEFF'.hexAColor;
color = '3767FEFF'.hexAColor;
```
